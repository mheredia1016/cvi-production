# ProductionOS v9.2 — Garment Order Date Selector

The S&S garment order now has its own date selector independent of the main
Production Date.

## New controls

- **Today**
- **Yesterday**
- **Use Production Date**
- Any custom calendar date

The chosen date controls:

- Build / Refresh Draft
- Refresh All S&S Inventory
- Save Draft Changes
- Mark Draft Reviewed
- Refresh individual S&S lines
- Print Out-of-Stock / Buy Elsewhere Report
- Future S&S submission for that draft

The selection is remembered in the browser.

## Same-day workflow

1. Set the main **Production Date** to today.
2. Preview and Shadow Import today's orders.
3. In **S&S Garment Order**, click **Today**.
4. Click **Build / Refresh Draft**.
5. Click **Refresh All S&S Inventory**.

If today's orders have not yet been imported, ProductionOS displays a message
explaining that no garment lines were found for that date.

## Previous-day workflow

Click **Yesterday** in the S&S section. The main Production Date can remain on a
different date without changing the garment purchase draft.
